\# SVAR: Euro Area Monetary Policy and Yield Curve



This repository contains the R code for estimating a structural VAR (SVAR) and generating figures and tables for the seminar paper on euro area monetary policy and the yield curve.



\## Structure



\- `SVAR.R`: Main R script that loads the data, estimates the SVAR, and exports figures and tables.

\- `Data/`: Contains dataset used in the analysis from ECB and Eurostat.

\- `Figures/`: Output figures generated by `SVAR.R`.

\- `Tables/`: Output tables generated by `SVAR.R`.



\## How to run



1\. Open `SVAR.R` in R or RStudio.

2\. Run the script. It will export figures to `Figures/` and tables to `Tables/`.
